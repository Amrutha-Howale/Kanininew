HTML ELEMENTS:
1) <html></html>: The <html> HTML element represents the root (top-level element) of an HTML document,
	so it is also referred to as the root element. All other elements must be descendants of this element.
2) <head></head>: The <head> HTML element contains machine-readable information (metadata) about the document,
	 like its title, scripts, and style sheets.
3) <link></link>: The <link> HTML element specifies relationships between the current document 
	and an external resource. This element is most commonly used to link to CSS.
4) <style></style>: The <style> HTML element contains style information for a document, or part of a document. 
	It contains CSS, which is applied to the contents of the document containing the <style> element.
5) <body></body>: The <body> HTML element represents the content of an HTML document. 
	There can be only one <body> element in a document.
